# Assistency API

A modern and scalable technical assistance management API built with Node.js, Express, MongoDB, and Socket.IO.

This project was designed to power real-world repair shops and technical support environments.

## Features

- Service order management
- Internal real-time chat
- AI-powered assistant system
- Role-based authentication
- Inventory and product control
- PDF generation
- Audit logging
- Semantic AI engine

## Technologies

- Node.js
- Express.js
- MongoDB
- Mongoose
- Socket.IO
- JWT
- Zod
- Swagger

## AI System

Custom-built semantic AI engine with:

- tokenization
- semantic relations
- contextual weighting
- adaptive confidence
- auto learning
- dynamic sentence generation

## Installation

```bash
git clone https://github.com/WandersonMorais02/api_assistency.git
```

```bash
yarn install
```

Create `.env`:

```env
PORT=3333
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
JWT_EXPIRES_IN=7d
```

Run development:

```bash
yarn dev
```
